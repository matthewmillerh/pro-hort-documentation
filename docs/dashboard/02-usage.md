# Dashboard Usage Guide

## Introduction

The Pro-Hort Dashboard provides a user-friendly interface for managing Pro-Hort operations. This guide covers the main features and how to use them.

## Main Components

### 1. Navigation

The main navigation is located in the sidebar. Use it to navigate between different sections:
- **Home** - Dashboard overview and quick stats
- **Cultivars** - Manage plant cultivars
- **Breeders** - Manage breeder information
- **Reports** - View and manage reports
- **Settings** - User preferences and system settings

### 2. Header

The header contains:
- Current section title
- User profile dropdown
- Logout button
- Search functionality

## Features

### Cultivars Management

Managing cultivar information:

1. Navigate to **Cultivars** section
2. View the cultivar table with all entries
3. Click on a cultivar to view details
4. Use the **Add New** button to create a new cultivar
5. Edit inline or open the edit dialog

**Available Fields:**
- Cultivar Name (required)
- Parentage
- Year Released
- Description
- Status (Active/Inactive)

### Breeders Management

Managing breeder information:

1. Navigate to **Breeders** section
2. View all breeders in the table
3. Add new breeders using the **Add New** button
4. Update breeder information as needed

### Reports

Viewing and exporting reports:

1. Navigate to **Reports**
2. Filter reports by type, date range, or status
3. Click on a report to view details
4. Export reports in various formats (PDF, CSV, Excel)

## Filters and Search

### Using Filters

The FiltersBar component provides filtering options:

1. Click on filter fields
2. Select filter criteria
3. Click **Apply** to apply filters
4. Click **Clear** to remove all filters

### Search

Use the search field to quickly find items:
- Search by name
- Search by ID
- Partial matches are supported

## Data Tables

### Features

- **Pagination** - Navigate through large datasets
- **Sorting** - Click column headers to sort
- **Selection** - Check boxes to select multiple items
- **Actions** - Right-click or use action menu for bulk operations

### Bulk Operations

1. Select multiple items using checkboxes
2. Click the action menu
3. Choose desired action (Delete, Export, Archive, etc.)

## Forms and Dialogs

### Creating New Items

1. Click **Add New** button
2. Fill in required fields (marked with *)
3. Enter optional information
4. Click **Save** or **Cancel**

### Editing Items

1. Click on an item in the table
2. Click **Edit** button
3. Modify the fields
4. Click **Save** to confirm changes

## Tips and Tricks

- **Keyboard Shortcuts**: Use Tab to navigate forms quickly
- **Quick Export**: Right-click any table and select "Export"
- **Favorites**: Mark frequently used views as favorites for quick access
- **Auto-save**: Draft changes are auto-saved every 30 seconds
- **Undo/Redo**: Use Ctrl+Z / Ctrl+Y to undo/redo actions

## Common Tasks

### Adding a New Cultivar

1. Go to **Cultivars** → **Add New**
2. Enter cultivar name (required)
3. Add parentage information (optional)
4. Set status to "Active"
5. Click **Save**

### Generating a Report

1. Go to **Reports** → **New Report**
2. Select report type
3. Choose date range
4. Click **Generate**
5. Export if needed

### Exporting Data

1. Select items or entire table
2. Right-click → **Export**
3. Choose format (CSV, Excel, PDF)
4. Click **Download**

## Troubleshooting

### Data Not Loading

- Refresh the page
- Clear browser cache
- Check your internet connection
- Verify server is running

### Form Submission Failed

- Check all required fields are filled
- Look for validation error messages
- Try clearing browser cache and cookies

### Performance Issues

- Close unused browser tabs
- Clear application cache
- Reduce the number of visible rows in tables
